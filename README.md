# AI Shopping Agent

An AI-powered shopping assistant that helps users find and compare products using natural language queries.

## Features

- Natural language product search
- Price comparison
- Product recommendation
- AI-powered decision assistance

## Technologies

- Python
- LLM APIs
- Web scraping / APIs
- Pandas
- Flask

## Example Query

User: "Find the best laptop under ₹80000 for machine learning"

Agent:
- Extracts requirements
- Searches products
- Compares specifications
- Returns best recommendations

## Future Improvements

- Multi-store price comparison
- Personalized recommendations
- Browser extension integration
